## 🆘 SUPPORT & TROUBLESHOOTING

### If News Feed is Empty:
1. Check NEWSDATA_API_KEY in `.env.local`
2. Verify key is active at newsdata.io/dashboard
3. Check free tier not exceeded (200/day)
4. See browser console (F12) for errors
5. Check network tab for API response

### If Chat Doesn't Respond:
1. Check internet connection
2. OpenAI/OpenRouter keys optional - should fallback
3. Check API limits not exceeded
4. Verify keys in environment variables
5. Restart dev server

### If Deployment Fails:
1. Run `npm run build` locally to test
2. Check all env vars set in Vercel
3. Verify GitHub permissions
4. Check `vercel.json` syntax
5. See Vercel build logs for errors
